FLV Extract CL

Python port of http://www.moitah.net/

It's pure python and has not external dependencies, it need python 2.6 or newer

Right now it supports (not much tested):
 - WAV writer
 - MP3 writer
 - AVI writer (vp6/screen, TODO: h263 picture size)
 - AAC writer
 - RAW H264 writer

build.sh: creates standalone executable called flvextract,
you need zip and cpio
